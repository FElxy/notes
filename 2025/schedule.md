## 🧭 前端高级成长计划（面向10年经验）

### ✅ 第二阶段：现代框架深度掌握（4\~6 周）

| **状态管理体系**                    | Redux / Pinia / Zustand / Jotai / Context 优劣分析       | 搭建中后台状态层模块                      |
| **路由系统**                      | 动态路由、懒加载、守卫、嵌套路由                                     | 自动路由插件开发                        |
| **渲染模式**                      | CSR / SSR / SSG / Hydration / Streaming              | 使用 Nuxt3 / Next.js 实战           |
